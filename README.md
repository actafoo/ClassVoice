# ClassVoice

수업 중 교사가 칭찬한 학생을 자동으로 추적하고, 포인트를 부여하는 웹앱입니다.

🌐 **[classvoice.vercel.app](https://classvoice.vercel.app)**

---

## ⚠️ 주의사항 및 면책 고지

### 개인정보 및 보안
- 이 프로젝트는 **개인정보보호법 및 아동·학생 개인정보 지침에 대한 법적 검증이 이루어지지 않은 도구**입니다. 학교·기관의 개인정보 처리 방침을 반드시 확인한 후 사용하세요.
- 수업 중 수집된 음성 텍스트는 설정한 LLM 제공사(Anthropic, OpenAI, Google, Upstage)의 외부 서버로 전송됩니다. **학생 발화가 포함되지 않도록 마이크를 반드시 교탁 위 교사 가까이에만 배치하세요.**
- API 키는 사용자 브라우저의 `localStorage`에만 저장되며 서버로 전송되지 않지만, 공용 PC 사용 시 사용 후 브라우저 데이터를 삭제하세요.
- 학생 이름 등 개인정보가 외부 LLM API로 전송되는 구조임을 인지하고 사용하세요.

### 기술적 한계
- LLM 분석 결과는 100% 정확하지 않을 수 있습니다. 결과 화면에서 반드시 수동 검토 후 포인트를 부여하세요.
- 현재 음성 인식은 브라우저의 Web Speech API(Google 서버 기반)를 사용합니다. **추후 외부 서버 의존도를 낮추기 위해 로컬 STT(온디바이스 음성 인식)로 전환할 예정입니다.**

## 주요 기능

- **실시간 음성 인식** — Web Speech API (한국어)로 수업 중 발화를 자동 수집
- **LLM 분석** — 수업 종료 후 전체 녹취를 LLM에 1회 전송해 칭찬받은 학생 추출
- **수동 보정** — 결과 화면에서 감지 누락/오감지 학생을 직접 체크/해제
- **포인트 자동 부여** — [Grownd](https://growndcard.com) 포인트 API 연동

## 지원 LLM

| 제공사 | 모델 |
|--------|------|
| Anthropic | claude-haiku-4-5 |
| OpenAI | gpt-4o-mini |
| Google | gemini-2.0-flash |
| Upstage | solar-pro |

## 사용 방법

### 1. 설정
- **LLM 설정**: 사용할 LLM 제공사와 API 키 입력
- **포인트 API 설정**: Grownd API 키, 클래스 ID, 칭찬 1회당 포인트 수 입력
- **학생 명단**: `출석번호 이름` 형식으로 한 줄씩 입력
  ```
  1 김지수
  2 이민준
  3 박서연
  ```

### 2. 수업 녹음
- **마이크는 교탁 위 교사 가까이에만 배치** — 학생 발화가 섞이지 않도록 주의
- **녹음 시작** 버튼을 눌러 수업 진행 (40분 타이머 / 진행바 표시)
- 음성은 자동으로 텍스트로 수집되며, 브라우저 5분 제한은 자동 재시작으로 우회

### 3. 결과 확인 및 포인트 부여
- 수업 종료 후 LLM이 칭찬받은 학생을 자동 감지
- 감지 목록에서 체크/해제로 수동 보정 가능
- **포인트 부여** 버튼으로 선택된 학생에게 일괄 포인트 지급

## 기술 스택

- 순수 HTML + CSS + JavaScript (빌드 도구 없음)
- 서버 없음, DB 없음
- 설정값은 브라우저 `localStorage`에 저장 (새로고침 유지)

## 파일 구조

```
index.html   — HTML 구조
styles.css   — 스타일
app.js       — 앱 로직
```

## 로컬 실행

별도 설치 없이 `index.html`을 Chrome에서 열면 됩니다.  
음성 인식은 **Chrome** 브라우저에서만 동작합니다.
